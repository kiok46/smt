### How to run via expo?

Please do the following:
```
npm install exp --global
git clone https://github.com/kiok46/smt.git
cd smt
npm install
```

Method: 1
```
exp start
scan the QR-code and test it on real device
```

Method: 2

`Open the expo app on your device and open the project`
